# 📖 Pokédex - Desafio DIO

Este projeto foi desenvolvido como parte do **Desafio Pokédex** proposto pelo DEV Renan na plataforma **Digital Innovation One (DIO)**.  
O objetivo é criar uma aplicação que consome a **PokéAPI** e exibe uma lista interativa de Pokémon, simulando uma Pokédex digital.

---

## 🚀 Tecnologias utilizadas

- **HTML5** → Estrutura da aplicação  
- **CSS3** → Estilização e responsividade  
- **JavaScript (ES6+)** → Lógica e integração com API  
- **PokéAPI** → Fonte de dados dos Pokémon  

---

## 🎯 Objetivos do desafio

- Consumir dados da **PokéAPI**.  
- Exibir uma lista de Pokémon com informações básicas (nome, imagem, número).  
- Implementar paginação para carregar mais Pokémon sob demanda.  
- Criar uma interface amigável e responsiva.  

---

## 🖼️ Demonstração

A aplicação exibe os Pokémon em cards, mostrando:  
- Nome  
- Número na Pokédex  
- Sprite oficial
